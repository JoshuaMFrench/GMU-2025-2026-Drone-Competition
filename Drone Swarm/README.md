The Drone Hacking Challenge:

Your Objectives: 
1. Successfully attack at least one of the drones in the swarm using your ARP spoofing and command spoofing skeleton script and make it execute up to 2 extra commands after you have gained control. 
2. Successfully attack a second drone in the swarm after reconfiguring scripts and make it execute up to 2 extra commands.

Guidlines:
You will be required to configure the IP and MAC addresses of the drones correctly in the skeleton scripts. 
Each team will be awarded 4 points for the successful takeover of 1 drone. 
Each command (up to two commands) following the drone takeover will result in 2 points for each command (4 points total). 
If you successfully hack, takeover and run 2 commands with one of the drones in the drone swarm, you will be awarded 8 points. 
If you are able to successfully attack, takeover, and command BOTH drones, you can earn up to 16 points (8 per drone). 
If the team is unable to hack either drone, there will be no points awarded.
