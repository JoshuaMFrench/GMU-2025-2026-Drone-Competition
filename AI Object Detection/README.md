You can use whatever computer vision model you want to so long as you detect all of the shape color combinations individually. Your code should only interact when presented with a circle, while ignoring triangles and squares.

Please download the raw image data from the folders provided. You may use our YOLO V8 labels to train your own YOLO V8 model if you wish, but again you can do whatever. It would be interesting to compare differences in performance between models, so we thought to give teams both the raw data and our labels so more adventurous teams could experiment.
